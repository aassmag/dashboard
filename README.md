# Интерактивный бизнес-дашборд для E-commerce

## Цель проекта
Создать единый источник правды (single source of truth) для мониторинга ключевых метрик онлайн-продаж. Дашборд позволяет быстро отвечать на вопросы:
*   Какая динамика выручки и количества заказов?
*   Какие регионы и категории товаров показывают лучший рост?
*   Какова конверсия пользователей по этапам воронки?
*   Каковы ключевые показатели по доставке и удовлетворенности клиентов?

## Источники данных и подготовка
*   **Источник:** Открытый датасет "Brazilian E-Commerce" с Kaggle.
*   **Таблицы:** `orders`, `customers`, `order_items`, `products`, `geolocation`.
*   **Подготовка:** Написаны SQL-запросы для соединения таблиц, агрегации и расчета метрик. Данные очищены от дубликатов и аномалий 
    *   скрипты в папке `sql_queries/`.

## Метрики на дашборде
Дашборд состоит из 3-х ключевых секций:

### 1. **Общие KPI (Key Performance Indicators)**
*   Выручка (Revenue) — текущий месяц vs предыдущий (с динамикой)
*   Количество заказов (Orders)
*   Средний чек (AOV)
*   Количество активных покупателей

### 2. **Анализ по измерениям (Dimensions)**
*   **География:** Карта продаж и топ-10 городов по выручке.
*   **Время:** Тренд выручки по неделям/месяцам (линейный график).
*   **Категории товаров:** Топ-10 категорий по выручке и количеству заказов (столбчатые диаграммы).
*   **Статус доставки:** Доля заказов с разным статусом (доставлен, в пути, отменен).

### 3. **Воронка конверсии**
Визуализация основных шагов пользователя:
    Просмотр сайта → Добавление в корзину → Оформление заказа → Оплата

## Интерактивность и фильтры
Дашборд позволяет:
*   Выбрать период анализа.
*   Фильтровать данные по **региону**, **категории товара**, **статусу заказа**.
*   Кликнуть на любой элемент (например, столбец с категорией), чтобы применить фильтр ко всем визуализациям дашборда (кросс-фильтрация).

## Технологии
*   **BI-инструмент:** Tableau Public
*   **Язык запросов:** SQL (для подготовки данных)
*   **Визуализации:** Карты, линейные графики, столбчатые диаграммы, KPI-карточки, круговая диаграмма.

## Инсайты, обнаруженные с помощью дашборда
1.  **Сезонность:** Пик продаж наблюдается в ноябре (предположительно, Black Friday).
2.  **Региональная концентрация:** 80% выручки генерируют 3 крупнейших штата.
3.  **Проблемы с доставкой:** Высокий процент заказов с статусом "доставка просрочена" в определенных регионах требует внимания логистического отдела.

## Структура репозитория
- `dashboard/` — файл проекта Tableau (`.twbx`) или Power BI (`.pbix`).
- `sql_queries/` — SQL-скрипты для извлечения и трансформации данных.
- `screenshots/` — скриншоты финального дашборда.
- `data/` — примеры исходных данных (опционально).
